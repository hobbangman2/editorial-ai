# User Journeys: editorial-ai

## 0. Guest Journey (Open Access)
목표: 로그인 없이 사설을 탐색하고 서비스의 가치를 경험함.

1.  **진입**: 웹사이트 접속 또는 앱 설치 후 첫 화면.
2.  **콘텐츠 탐색**: '트렌딩 사설' 또는 '최신 뉴스 사설' 리스트 조회.
3.  **사설 읽기**: 원하는 사설을 선택하여 본문 및 어휘 읽기.
4.  **인터랙션 시도**: '좋아요' 누르기, '댓글' 작성, 또는 '받아쓰기 시작' 클릭 시 로그인/회원가입 팝업 노출.
5.  **동기 부여**: "로그인하고 나만의 맞춤 사설과 보상을 확인하세요!" 포인트 노출.

## 1. Student Journey (Member Loop)
목표: 매일 새로운 사설을 읽고 받아쓰며 문해력을 키움.

1.  **진입**: 앱 알림 또는 실행.
2.  **온보딩/프로필 설정 (최초 1회)**: 닉네임, 아바타(이모지), 관심 주제(아이돌, 스포츠 등), 학습 목표(일 1회 등) 설정.
3.  **광고 확인**: 메인 화면의 교육 목적 펀딩 배너 또는 상단 광고 노출 확인.
4.  **콘텐츠 선택**: 맞춤형 사설 또는 퀘스트 사설 선택.
5.  **받아쓰기 수행**: 실시간 오답 체크 및 ASMR 모드.
6.  **완료 및 인터랙션**: 
    -   좋아요/댓글로 의견 표현.
    -   학습 결과 화면에서 네이티브 광고(펀딩 지원) 확인.
7.  **성취**: 학습 달력 스탬프 및 부모님 알림.

## 2. Parent Journey (Management & Support)
목표: 자녀에게 학습 동기를 부여하고 성장을 확인 함.

1.  **퀘스트 설정**: "이번 주 사설 5개 완료 시 편의점 쿠폰" 등 퀘스트 생성.
2.  **모니터링**: 대시보드를 통해 자녀가 읽고 있는 사설의 주제와 오답률 확인.
3.  **격려**: "오늘 사설 내용이 어렵던데 잘했어!" 등 따뜻한 응원 푸시 전송.
4.  **보상 승인**: 자녀의 퀘스트 완료 알림 수신 후 '보상 완료' 클릭.

## 3. Admin Journey (The Guardian)
목표: AI 콘텐츠의 품질을 유지하고 수익 모델을 관리함.

1.  **프롬프트 튜닝**: 학습 효과를 높이기 위해 사설 생성 프롬프트(논조, 난이도) 수정.
2.  **품질 검수**: 생성된 사설 중 부적절한 내용이 있는지 필터링 및 수정.
3.  **사용자 관리**: 휴먼 서비스 철학에 따른 무료/유료 비율 조정 및 프로모션 관리.
