# Epics & User Stories: editorial-ai

## Epic 1: Multi-Stage AI Pipeline
**목표**: 뉴스 분석부터 사설 생성, 학습 요소 추출까지 이어지는 AI 핵심 로직 구현.

- **Story 1.1: 주제 선정 (Topic Selection)**
    - *As a* 시스템, *I want to* 최신 뉴스를 가져와 교육적 가치와 학생 관심사를 반영한 주제를 선정하고 싶다.
    - *AC*: 뉴스 API 연동, Prompt A 사용, JSON 결과 반환.
- **Story 1.2: 사설 본문 생성 (Editorial Generation)**
    - *As a* 시스템, *I want to* 선정된 주제와 난이도 설정을 기반으로 사설 본문을 작성하고 싶다.
    - *AC*: Prompt B 사용, 수준별(LEVEL 1~3) 최적화, TTS용 텍스트 생성.
- **Story 1.3: 학습 요소 추출 (Extraction)**
    - *As a* 시스템, *I want to* 생성된 사설에서 핵심 어휘와 퀴즈를 자동으로 뽑아내고 싶다.
    - *AC*: Prompt C 사용, 단어 뜻 및 예문 포함 JSON 반환.

## Epic 2: 사용자 계정 및 환경 (Auth & Profile)
**목표**: 비로그인 접근 제어 및 상세 개인화 환경 구축.

- **Story 2.1: 게스트 접근 제어**
    - *As a* 비로그인 사용자, *I want to* 사설을 읽어보고 서비스의 가치를 확인하고 싶다.
    - *AC*: 사설 리스트/본문 조회 가능, 인터랙션 시 로그인 유도 팝업.
- **Story 2.2: 상세 프로필 설정**
    - *As a* 회원, *I want to* 닉네임, 아바타(이모지), 관심 주제, 목표를 설정하여 나만의 학습 환경을 만들고 싶다.
    - *AC*: 프로필 수정 페이지 구현, 신문사 선호도(The Guardian 등) 저장.
- **Story 2.3: 소셜 인터랙션 (좋아요/댓글)**
    - *As a* 회원, *I want to* 사설에 좋아요를 누르고 댓글을 달아 다른 사용자들과 소통하고 싶다.
    - *AC*: DB 연동, 실시간 반영 UI.

## Epic 3: 디지털 학습 경험 (Learning UI)
**목표**: 태블릿 펜슬 및 키보드 기반의 고품질 받아쓰기 기능 제공.

- **Story 3.1: 펜슬 모드 실시간 받아쓰기**
    - *As a* 학생 사용자, *I want to* 아이패드 펜슬로 사설을 따라 쓰며 실시간 피드백을 받고 싶다.
    - *AC*: Canvas/SVG 기반 필기 구현, OCR 연동, ASMR(종이 소리) 옵션.
- **Story 3.2: 키보드 모드 받아쓰기 (웹/앱 공통)**
    - *As a* 웹 사용자, *I want to* 키보드로 사설을 타이핑하며 오타 및 문맥 오류를 체크받고 싶다.
    - *AC*: 텍스트 대조 알고리즘 적용, 실시간 오답 강조.

## Epic 4: 관리자 및 수익화 (Admin & Ads)
**목표**: 운영 효율성 확보 및 서비스 지속 가능성을 위한 광고 적용.

- **Story 4.1: 멀티 프롬프트 관리 UI**
    - *As a* 관리자, *I want to* 서비스 운영 중에도 AI 기능별 프롬프트를 수정하고 배포하고 싶다.
    - *AC*: Prompt A, B, C 개별 수정 폼, 버전 이력 조회.
- **Story 4.2: 광고 및 펀딩 배너**
    - *As a* 관리자, *I want to* 펀딩 모집이나 교육 광고 배너를 특정 위치에 노출하고 싶다.
    - *AC*: 메인 상단 배너, 학습 완료 결과 페이지 네이티브 광고.

## Epic 5: 부모-자녀 학습 루프 (Rewards)
**목표**: 가족 간의 상호작용을 통한 학습 동기 부여.

- **Story 5.1: 부모 퀘스트 발급 및 보상**
    - *As a* 부모, *I want to* 자녀에게 보상을 걸고 학습 퀘스트를 부여하고 싶다.
    - *AC*: 퀘스트 생성 폼, 자녀 완료 알림 수신, 보상 지급 확인 처리.
